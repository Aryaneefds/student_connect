## 🚀 How to Run

### 1. Navigate to Project
```bash
cd campus_connect
```

### 2. Start the App

**Option A - Expo Go (Fastest)**
```bash
npx expo start
# Scan QR code with Expo Go app
```

**Option B - Android Emulator**
```bash
npm run android
```

**Option C - Web Preview**
```bash
npm run web
```

**Option D - Production Build**
```bash
eas build --platform android
```
